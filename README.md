# Microcontroller Projects (VŠB-TUO Summer School)

Collection of small projects developed during a 25-hour summer school focused on microcontroller programming at VŠB–TUO.

## Technologies used:
- Arduino Uno
- ESP32
- C / C++
## Topics covered
- Basics of embedded programming
- Working with digital inputs and outputs
- Serial communication
- Simple control logic
- Hardware–software interaction
## Project structure
Each folder contains an independent exercise or experiment completed during the course.

## What I learned
- Basics of embedded systems
- Development workflow for microcontrollers
- Writing and debugging low-level code
- Understanding interaction between software and hardware
## Possible future improvements
- Merge projects into larger systems
- Improve code organization
- Add documentation and wiring diagrams
- Extend projects with sensors, networking or digital displays
